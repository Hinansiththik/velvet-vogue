# 🛍️ Velvet Vogue - E-Commerce System

Velvet Vogue is a full-stack e-commerce web application built using **PHP, MySQL, HTML, CSS, and JavaScript**.

## 🚀 Features

### 👤 Customer
- Register & Login
- Browse products
- Add to cart
- Checkout system
- Order history
- Printable receipt
- Inquiry system

### 🛠️ Admin Panel
- Dashboard with statistics
- Manage products
- Manage orders
- Update order status
- Manage users
- Handle inquiries

## 🧰 Technologies Used
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP
